Uilerial v2

Professional React UI library — 30+ components, CSS Modules, TypeScript.

Install

npm i @techlestial/uilerial

Peer dependencies: react and react-dom >= 18.

Quick start

import { Button, Card, DataTable, ToastContainer } from "@techlestial/uilerial";

export function App() {
  return (
    <>
      <ToastContainer position="top-right" />
      <Card title="Hello">
        <Button variant="primary">Ship it</Button>
      </Card>
    </>
  );
}
